# Sound Files Directory Structure

Place your OGG sound files in this directory structure:

## Music Disc
sounds/untitled_disk_song.ogg

## Horror/Event Sounds
sounds/ted_lewis_fuck_you.ogg
sounds/errrrrr.ogg
sounds/forever.ogg

## Ambient Horror Sounds
sounds/ambient/whisper_1.ogg
sounds/ambient/whisper_2.ogg
sounds/ambient/static_noise.ogg
sounds/ambient/heartbeat.ogg
sounds/ambient/breathing.ogg
sounds/ambient/door_creak.ogg
sounds/ambient/footsteps_horror.ogg
sounds/ambient/laugh_distant.ogg
sounds/ambient/scream.ogg
sounds/ambient/jumpscare.ogg

## Event-Specific Sounds
sounds/events/event_ijoin.ogg
sounds/events/event_watching.ogg
sounds/events/event_glitch.ogg

## Requirements:
- All files must be in OGG Vorbis format
- Recommended sample rate: 44100 Hz
- Recommended bit rate: 128-192 kbps
- Mono or stereo (stereo preferred for immersion)

## Music File (untitled_disk_song.ogg):
- Length: 79 seconds
- Will play when the music disc is used in a jukebox
- Stream: true (for better performance)

## Note:
These files are NOT included in the repository.
You must provide your own sound files with these exact names.
